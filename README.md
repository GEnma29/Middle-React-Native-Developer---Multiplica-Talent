# Middle-React-Native-Developer---Multiplica-Talent

## clonar repositorio

```bash
git clone https://github.com/GEnma29/Middle-React-Native-Developer---Multiplica-Talent.git
```

## instalar las dependencias 
```bash
yarn
```
or 
```bash
npm i
```

## run project 
### ios
```bash
yarn ios
```

### ios
```bash
yarn android
```

### Stack
- React Native CLI
- React Native Elements 
- SWR
- Axios
